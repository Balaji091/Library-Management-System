Help Menu

To search for books, use the search bar located at the top of the library management system interface. Enter keywords, titles, authors, or ISBN numbers to find relevant books.

To borrow books, select the desired book and click on the "Borrow" button. Make sure you have your student ID card ready for verification purposes.

To renew books, go to your account settings and find the "Renew" option next to the borrowed books. You can renew each book once unless it has been reserved by someone else.

To return books, visit the library and drop off the books at the return counter. Alternatively, you can return them through the library management system by selecting the "Return" option next to each borrowed book.
